== Description ==

Comment2QQ 能把WP上所有评论实时发送到QQ群

== Installation ==

1. 上传到插件目录下
2. 输入 QQ群 官方 Webhook 链接
3. 尝试评论吧！评论内容会发送到链接指定QQ群

了解更多以及图文帮助：https://krunk.cn/kblog1624.html
